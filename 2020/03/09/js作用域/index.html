🌟作用域

  • 指变量的合法使用范围；

  • 全局作用域：变量没有收到函数的约束，全局使用，如window, document对象

  • 函数作用域：只能在当前函数内使用，如上图

  • 块级作用域：ES6新增，{}代码块如if语句

  • 自由变量：变量被使用，当前作用域没有被定义，则向上一级作用域去找，直到找到为止，如全局也没有，则报错:xx is not defined。

🌟闭包

  • 定义：自由变量的查找，是在函数定义的地方，向上级作用域查找，不是在执行的地方。

  • 两个场景：

    • 函数作为参数被传递
    • 函数作为返回值被返回
    //example1
function g(){
    const a = 100;
    return function(){
        console.log(a);
    }
}
const a = 200;
let fn1 = g();
fn1(); //100

//example2
const b = 100;
function fn2(){
   console.log(b); 
}

function fn3(fn){
    const b = 200;
    fn2(); //100
}
fn3();

###🌟this

  • this指向的值，是函数执行的时候被确定的，不是在定义的时候被确定的。

  • 使用场景

    • 普通函数调用
    • 使用call/apply/bind，区别：call，参数很多项；apply：参数1项；bind：返回新的函数，不会马上执行，参数很多项。
    • 作为对象方法被调用
    • class方法中调用
    • 箭头函数的this取值：绑定this取值，为定义时上一层的this取值。
    //1普通函数
function fn(){
    console.log(this) 
}
fn()//window

//2apply call bind
fn.call({x:100}) //{x:100}
fn2 = fn.bind({x:100})
fn2();// {x:100}

//3对象方法被调用
const zhang = {
    name: 'zhang',
    sayHi(){
        console.log(this.name)
    },
    wait(){
        setTimeout(function(){
            console.log(this.name) // this === window
        },2000)
    }
}

//4箭头函数中被调用
const zhang = {
    name: 'zhang',
        sayHi(){
        console.log(this.name)
    },
    wait(){
        setTimeout(()=>{console.log(this.name)},2000) //this === zhang
    }
}

//5class中
class People{
    constructor(name){
        this.name = name;
    }
    sayHi(){
        console.log(this) 
    }
}
const zhang = new People('zhang');
zhang.sayHi() // zhang这一对象

###🌟手写一个bind函数

Function.prototype.newBind = function(){
    const args = Array.prototype.slice.call(arguments);
    //获取this
    const t = args.shift();
    //获取fn
    const self = this;
    return function(){
        return self.apply(t, args); 
    }
}

  • 隐藏数据，只提供api

    function creatCache(){
    const data = {};
    return {
        set: function(key, value){
            data[key] =  value;
        }, //逗号！
        get: function(key){
            return data[key]
        }
    }
}

const c = creatCache();
c.set(1,2);
c.get(1);
