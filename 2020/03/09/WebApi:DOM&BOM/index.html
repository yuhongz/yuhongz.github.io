🌟DOM

  • DOM本质是从html语言解析出来的树结构

  • DOM操作：

    • 获取DOM节点

      document.getElementById('') //元素
document.getElementsByTagName('') //集合
document.getElementsByClassName('') //集合
document.querySelectorAll('p') //集合
      • 节点property和节点attribute
      //property
const pList = document.getElementsByTagName('p');
const p = pList[0];
p.style.width = '100px';
p.className = '.red'
consoleo.log(p.nodeName);
consoleo.log(p.nodeType);
//attribute
p.setAttribute('data-name', 'xxx');
p.setAttribute('style', 'font-size:15px;');
      • 区别：property修改对象属性，不会体现在html结构中；attribute修改html属性，会改变html结构。
      • 相同点：两者都有可能引起DOM重新渲染。

    • DOM结构操作：

      //新增 插入
const div1 = document.getElementById('x');
const p = document.createElement('p');
p.innerHTML = 'p1';
div1.appendChild(p);
//移动已有节点，也可选中节点后，使用appendChild
//获取父节点
p.parentNode;//非函数！！！
//获取子节点
p.childNodes
//子节点类型筛选
Array.portotype.slice.call(p.childNodes).filter( child => {
    if(child.nodeType === 1) return true;
    else return false;
})
//删除子元素
div1.removeChild(p);

  • 🌟DOM性能优化🌟：DOM操作非常昂贵，会消耗CPU，避免频繁操作。

    • 缓存DOM查询结果。

    • 将频繁操作改为一次性操作，如创建文件片段。

      //1.缓存查询
//错误示范
for(let i = 0; i < document.getElementsByClassName('xx').length; i++){
    ...
}
//应将查询保存至变量

//2.一次性操作 创建文件片段
const frag = document.createDocumentFragment();
frag.appendChild(p1);
frag.appendChild(p2);
div.appendChild(frag);

ua.indexOf('Chrome') //判断浏览器是否为chrome
//screen
screen.width
screen.height

location.href
